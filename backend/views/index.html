User Routes

Method Path Description
POST /register Register a new user.
POST /login Login a user that's already registered.
GET /users To get all the users that are registered
GET /user/id Get a user by using the unique id.
DELETE /user/:id Delete a user by using it's unique id.
PUT /user/:id Update an existing user profile by using the the id.

Products routes

Method Path Description
GET /products Get all the products.
GET /product/:id Get a single product by using the unique id.
POST /product Add a product to the API.
PUT /product/:id Update a product by making use of the unique id.
DELETE /product/:id Delete a single product by using the id.
